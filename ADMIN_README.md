# פורטל אדמין - הוראות שימוש

## גישה לפורטל האדמין

1. פתח את הדפדפן וגש לכתובת: `http://localhost:5005/admin`
   - או: `http://localhost:5005/#admin`
   - או: `http://localhost:5005?admin=true`
2. הזן את הסיסמה (מוגדרת בקובץ `.env`)
   - הסיסמה הנוכחית: `SagiSamurai2024!@#Secure`
   - הסיסמה לא מופיעה בקונסול או בקוד - היא נשמרת בקובץ `.env` בלבד

## איך לערוך את התוכן

### 1. סעיף ראשי (Hero)
- ערוך את הכותרת הראשית, הכותרת המודגשת, התת-כותרת והכפתורים

### 2. תפריט עליון (Navbar)
- שנה את שם המותג וקישור ה-WhatsApp

### 3. יתרונות (Benefits)
- ערוך, הוסף או מחק יתרונות
- כל יתרון כולל כותרת ותיאור

### 4. שירותים (Services)
- ערוך, הוסף או מחק שירותים
- כל שירות כולל ID, כותרת ותיאור

### 5. תהליך עבודה (Process)
- ערוך את שלבי התהליך (4 שלבים)

### 6. תיק עבודות (Portfolio)
- הוסף, ערוך או מחק סרטוני YouTube
- כל סרטון דורש: קישור YouTube, כותרת וקטגוריה

### 7. המלצות (Testimonials)
- הוסף, ערוך או מחק המלצות
- כל המלצה כוללת: שם, תפקיד ותוכן

### 8. שאלות נפוצות (FAQs)
- הוסף, ערוך או מחק שאלות ותשובות

### 9. צור קשר (Contact)
- ערוך פרטי יצירת קשר: טלפון, אימייל, קישורי רשתות חברתיות

### 10. כותרות סעיפים (Sections)
- שנה את הכותרות והתת-כותרות של כל הסעיפים בדף

## שמירת שינויים

1. לאחר ביצוע שינויים, לחץ על כפתור **"שמור"** בחלק העליון
2. השינויים נשמרים ב-localStorage של הדפדפן
3. הדף הראשי יטען את השינויים אוטומטית

## תצוגה מקדימה

- לחץ על כפתור **"תצוגה מקדימה"** כדי לראות איך הדף נראה עם השינויים
- התצוגה המקדימה נפתחת בחלון נפרד

## הערות חשובות

⚠️ **שימו לב:**
- השינויים נשמרים ב-localStorage של הדפדפן
- אם תרצה לשמור את השינויים לצמיתות, תצטרך לעדכן את קובץ `public/site-data.json` ידנית
- בסביבת ייצור, מומלץ להוסיף backend לשמירת הנתונים במסד נתונים

## שינוי סיסמה

כדי לשנות את סיסמת האדמין:
1. פתח את הקובץ `.env` (אם לא קיים, העתק מ-`.env.example`)
2. שנה את הערך של `VITE_ADMIN_PASSWORD` לסיסמה הרצויה
3. הפעל מחדש את השרת (`npm run dev`)
4. **חשוב:** קובץ `.env` לא נשמר ב-Git (מופיע ב-.gitignore) - הסיסמה לא תופיע בקוד

## גיבוי נתונים

מומלץ לגבות את הנתונים באופן קבוע:
1. בפורטל האדמין, לחץ על כפתור "שמור"
2. פתח את כלי הפיתוח של הדפדפן (F12)
3. עבור ל-Application > Local Storage
4. העתק את התוכן של `site_data`
5. שמור אותו כקובץ JSON
